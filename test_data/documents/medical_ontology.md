# 医疗本体知识文档

## 1. 疾病分类

### 1.1 常见慢性病

**高血压（Hypertension）**
- 定义：收缩压 ≥ 140mmHg 或舒张压 ≥ 90mmHg
- 分级：
  - 1级：收缩压 140-159 / 舒张压 90-99
  - 2级：收缩压 160-179 / 舒张压 100-109
  - 3级：收缩压 ≥ 180 / 舒张压 ≥ 110
- 风险因素：年龄、遗传、肥胖、吸烟、高盐饮食

**2型糖尿病（Type 2 Diabetes）**
- 定义：空腹血糖 ≥ 7.0 mmol/L 或餐后2小时血糖 ≥ 11.1 mmol/L
- 并发症：糖尿病肾病、视网膜病变、神经病变、足部溃疡
- 控制指标：HbA1c < 7%，空腹血糖 4-7 mmol/L

## 2. 药物体系

### 2.1 降压药物

| 药物类别 | 代表药物 | 适应症 | 禁忌症 |
|---------|---------|--------|--------|
| ARB类 | 缬沙坦 | 高血压、心衰 | 妊娠、双肾动脉狭窄 |
| ACEI类 | 卡托普利 | 高血压、糖尿病肾病 | 妊娠、肾功能不全 |
| 钙通道阻滞剂 | 氨氯地平 | 高血压、心绞痛 | 心源性休克 |
| β受体阻滞剂 | 美托洛尔 | 高血压、心律失常 | 哮喘、二三度房室传导阻滞 |

### 2.2 降糖药物

- **二甲双胍**：一线用药，降低肝糖输出，适合超重患者
- **SGLT-2抑制剂**：降糖同时有心脏和肾脏保护作用
- **GLP-1受体激动剂**：减重效果好，降低心血管风险
- **磺脲类**：促进胰岛素分泌，低血糖风险较高

## 3. 治疗流程

### 3.1 高血压管理流程

1. **初诊评估** → 测量血压（多次）→ 评估心血管风险
2. **生活方式干预** → 减盐、戒烟、减重、运动
3. **药物治疗** → 根据分级和并发症选药
4. **监测随访** → 每月监测血压，每半年评估靶器官损害
5. **达标评估** → 血压 < 140/90 mmHg 为达标

### 3.2 临床决策规则

- 规则1：高血压合并糖尿病 → 首选 ARB 或 ACEI
- 规则2：高血压合并心衰 → 首选 β受体阻滞剂 + 利尿剂
- 规则3：空腹血糖 > 13.9 mmol/L → 立即启动胰岛素治疗
- 规则4：收缩压 > 180 且有靶器官损害 → 高血压急症处理

## 4. 实体关系

- 高血压 **导致** 心脏肥大、肾功能损害、脑卒中
- 糖尿病 **并发** 高血压（约50%患者）
- 二甲双胍 **治疗** 2型糖尿病
- ARB类药物 **抑制** 肾素-血管紧张素系统
- 胰岛素 **降低** 血糖水平
