# 营销策略与客户运营手册

## 1. 客户分层体系

### 1.1 客户生命周期价值分层（CLV Segmentation）

**S级战略客户**
- 定义：年度合同额 > 500万元，合作年限 ≥ 3年
- 数量：约占总客户数 5%，贡献 GMV 的 45%
- 管理：专属客户成功经理，季度 QBR（业务回顾），定制化 SLA
- 代表客户：华为供应链、比亚迪采购部、宁德时代

**A级重点客户**
- 定义：年度合同额 100-500万元
- 管理：每月巡访，专属售后通道，优先资源调配
- 占比：客户数 15%，GMV 贡献 35%

**B级成长客户**
- 定义：年度合同额 20-100万元，有明显增长趋势
- 管理：季度回访，标准服务包，运营团队跟进
- 重点目标：通过产品深化和交叉销售提升至 A 级

**C级长尾客户**
- 定义：年度合同额 < 20万元
- 管理：线上自助服务，自动化营销触达
- 转化策略：行为数据触发，精准运营

## 3. 产品线与定价策略

### 3.1 产品矩阵

**基础版（Starter）**
- 目标客群：中小企业，员工 < 200人
- 功能：库存管理、基础采购、供应商管理
- 定价：¥ 2,880元/月，最少签 12个月
- 市场渗透目标：Y+1 新增 500家

**专业版（Professional）**
- 目标客群：成长型企业，员工 200-1000人
- 功能：全功能供应链、智能预测、多仓管理、API集成
- 定价：¥ 9,800元/月起，按模块计费
- ARR 贡献目标：¥ 1.2亿

**企业版（Enterprise）**
- 目标客群：大型集团企业，员工 > 1000人
- 功能：私有化部署 / 混合云、定制开发、专属运维
- 定价：¥ 50万+/年，根据规模议价
- 重点行业：汽车制造、消费电子、医药流通

### 3.2 交叉销售与增值服务

- **数据分析模块**：¥ 1,500元/月，NPS 78分，复购率 92%
- **供应商金融服务**：账期融资、保理，佣金收入 0.8%-1.5%
- **培训认证体系**：¥ 3,800元/人，年培训 2,000+人次
- **实施咨询服务**：¥ 800元/人天，平均项目周期 45天

## 4. 销售漏斗与转化模型

```
认知层（Awareness）   → 月触达 500万人次
兴趣层（Interest）    → 月 UV 25万（转化率 5%）
考虑层（Consideration）→ 月有效线索 3,200条（转化率 1.28%）
意向层（Intent）      → 月 Demo 请求 650次（转化率 20.3%）
评估层（Evaluation）  → 月进入 POC 120个（转化率 18.5%）
决策层（Decision）    → 月新签合同 45个（转化率 37.5%）
```

**漏斗关键指标**
- 平均销售周期：B 级客户 45天，A 级客户 90天，S 级客户 180天
- 平均合同额：B 级 ¥ 35万，A 级 ¥ 185万，S 级 ¥ 680万
- Win Rate（胜率）：公开招标 38%，直销 52%，渠道 44%

## 5. 品牌与市场定位

**目标市场份额**：

## 6. 客户成功与留存策略

**Onboarding 旅程（前90天）**
- Day 1-7：技术配置、数据迁移、基础培训
- Day 8-30：核心场景上线、关键用户认证
- Day 31-90：业务深化、自动化规则配置、首次 QBR

**续约与扩张运营**
- 健康度评分模型：登录频率（30%）+ 功能使用深度（40%）+ 业务效果（30%）
- 健康度 < 60 分触发 CSM 预警介入
- 年度续约率目标：S级 98%，A级 93%，B级 85%

**客户流失预警规则**
- IF 连续 14天无登录 AND 上月工单 > 3 THEN 触发客户挽回流程
- IF 合同到期前 90天 AND 健康度 < 70 THEN 启动续约专项
