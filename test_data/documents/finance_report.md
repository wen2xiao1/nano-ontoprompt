## 财务指标

- **流动比率** = 流动资产 / 流动负债 = 5,750 / 3,050 = **1.88**（正常范围 1.5-2.5）
- **资产负债率** = 负债 / 资产 = 6,050 / 15,250 = **39.7%**（健康）
- **毛利率** = 毛利润 / 营业收入 = 4,200 / 12,000 = **35%**
- **净利率** = 净利润 / 营业收入 = 1,650 / 12,000 = **13.75%**
- **ROE** = 净利润 / 所有者权益 = 1,650 / 9,200 = **17.9%**

## 财务规则

1. **会计基础规则**：资产 = 负债 + 所有者权益
2. **坏账计提规则**：应收账款逾期90天计提50%，逾期180天计提100%
3. **折旧政策**：固定资产采用直线法，使用年限5-20年，残值率5%
4. **存货计价**：采用加权平均法计算成本
5. **关联交易规则**：与关联方交易须在附注中披露，按公允价值定价
